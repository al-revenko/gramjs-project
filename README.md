# GramJS project

### Первый запуск

Установка зависимостей:

```bash
npm i
```

В корне проекта создайте файл `.env` и скопируйте в него переменные из файла [env.example](./.env.example). Задайте нужные значения переменным:

```env
TG_API_ID="my-api-id"
TG_API_HASH="my-api-hash"
```

Запустите приложение:

```bash
npm run start
```
